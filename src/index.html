ex1) ループなし

標準ではループする処理ですが、オプションのloopをfalseにすると無効にできます。

<script>
$(function(){
	$('#example1_ex1').smplslider({
		loop: false
	});
});
</script>
<!-- HTML省略 -->

ex2) ループの種類を変更

標準のループは最後のスライドの次に最初のスライドへ戻るだけですが、オプションのrouletteLoopをtrueにするとルーレットのようなループになります。 

<script>
$(function(){
	$('#example1_ex2').smplslider({
		loop: false
	});
});
</script>
<!-- HTML省略 -->

ex3) マウスオーバーでスライドショー停止

オプションのhoverStopをtrueにするとスライダーとカーソルが重なっている状態のときにスライドショーを停止させます。
カーソルの重なりが解除されると再びスライドショーが再開されます。 

<script>
$(function(){
	$('#example1_ex3').smplslider({
		hoverStop: true
	});
});
</script>
<!-- HTML省略 -->

ex4) スライドショーの待機時間を変更する

オプションのintervalを使うことで自動的にスライドするまでの待機時間（ミリ秒で指定）を調整できます。 

<script>
$(function(){
	$('#example1_ex4').smplslider({
		interval: 1000
	});
});
</script>
<!-- HTML省略 -->

ex5) 初回のスライドショーが始まるまでの待機時間を設定

オプションのslideShowDelayでスライドショーが始まるまでの遅延時間（ミリ秒）を設定することができます。
値をfalseにするとこのオプションは無効になり、intervalの値による待機時間が適用されます。 

<script>
$(function(){
	$('#example1_ex5').smplslider({
		slideShowDelay: 5000
	});
});
</script>
<!-- HTML省略 -->

ex6) スライドの方向を変更

オプションのorientationでスライドの方向を縦（vertical）と横（horizontal）で変更できます。

<script>
$(function(){
	$('#example1_ex6').smplslider({
		orientation: 'vertical'
	});
});
</script>
<!-- HTML省略 -->
